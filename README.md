# Mind - Base de Conhecimento Estruturada

Este repositório contém um **Cérebro Digital** organizado como um vault do Obsidian, focado em gestão financeira, Agronegócio e Fundos de Investimento (FIDC, FIAGRO, FII).

## 🚀 Visão Geral

O projeto automatiza e organiza o conhecimento técnico de um gestor de recursos, utilizando uma estrutura de pastas rigorosa, índices dinâmicos e uma IA (Gemini CLI) para auxiliar na manutenção e expansão das notas.

## 📂 Estrutura de Pastas

- `Mind/Conhecimento/`: O núcleo técnico (Mercado, Crédito, Agro, Fundos, etc.).
- `Mind/daily/`: Diário de bordo e registros diários.
- `.claude/skills/`: Instruções especializadas para agentes de IA operarem no vault.
- `GEMINI.md`: Regras de governança para o agente Gemini.
- `AGENTS.md`: Diretrizes gerais para agentes e manuais de operação.

## 🛠️ Tecnologias e Ferramentas

- **Obsidian**: Interface principal de leitura e escrita.
- **Gemini CLI**: Automação e geração de conteúdo técnico.
- **Markdown**: Formato universal das notas.
- **UTF-8**: Encoding obrigatório para suporte a acentos.

## 📏 Regras de Contribuição

Antes de criar ou editar notas, consulte:
1. `Mind/REGRAS_CONHECIMENTO.md`: Padrões de escrita, links e hashtags.
2. `.claude/skills/conhecimento.md`: Fluxos de trabalho da IA.
